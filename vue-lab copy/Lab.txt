1. Опишіть кроки, які ви виконали для інсталяції та налаштування Tailwind CSS.
-У кореневій директорії проєкту встановив Tailwind CSS разом з офіційним плагіном для Vite
-Відкрив файл конфігурації Vite та додав у нього Tailwind як окремий плагін
-Створив окремий CSS-файл у папці проєкту та підключив до нього Tailwind
-Додав імпорт цього CSS-файлу у головний файл запуску застосунку, щоб Tailwind працював у всіх компонентах
-Перезапустив сервер розробки, щоб Tailwind активувався й почав застосовувати стилі в проєкті
2. Реалізація умовних стилів у UserCard.vue за допомогою Tailwind
-У компоненті UserCard.vue я видалив увесь <style scoped> і замінив його на utility-класи Tailwind
-Щоб реалізувати умовну зміну стилів , використав Vueʼs :class + Tailwind-класи
3. Переваги та недоліки
Переваги:
-Не потрібно писати окремі CSS класи 
-Усі стилі видно прямо в шаблоні
-Атоматична темна тема через dark
-Швидша розробка 
Недоліки: 
-Важче читати
-Весь дизанй зосереджений прямо в шаблоні, що інколи не підходить для великих команд
-HTML буде виглядати забитим через велику кількість класів
4. Темна тема(бонус)
-додав стилі з перфіксом dark у UserCard.vue:
    :class="isActive 
      ? 'border-green-500 bg-green-50 dark:bg-green-900 dark:border-green-700' 
      : 'border-gray-300 bg-white dark:bg-gray-800 dark:border-gray-600'"
